The Tax Infrastructure

The tax code is progressive. The enforcement mechanism is not. That is not a contradiction — it is the architecture. The United States federal tax system imposes a higher marginal rate on higher incomes. This is law. What is also law, but less discussed, is the mechanism that converts the statutory progressivity into an actual redistribution of resources: the enforcement infrastructure that identifies underpayment, pursues it, and collects it. That infrastructure — the IRS examination and collection function — has been structurally degraded to the point where the progressivity of the tax code is now largely fictional.
